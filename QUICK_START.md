# 🚀 快速开始 - AI 小狗生成器

## 30秒上手指南

### 第一步：设置 API Key
打开 `ai-generator.js` 文件，在第 6 行找到：
```javascript
apiKey: 'YOUR_OPENAI_API_KEY_HERE',
```
将 `'YOUR_OPENAI_API_KEY_HERE'` 替换为你的实际 OpenAI API Key

> **获取 API Key**：访问 https://platform.openai.com/api-keys
> 
> **详细设置指南**：查看 `API_KEY_SETUP.md`

### 第二步：打开网页
在浏览器中打开 `index.html`

### 第三步：点击 AI 生成按钮
在预览面板下方找到 **"🤖 AI生成"** 按钮并点击

### 第四步：开始对话！
在输入框中描述你想要的小狗，例如：

```
我想要一只戴着牛仔帽和太阳镜的酷狗
```

### 第五步：查看结果
AI 会自动为你的小狗选择合适的配件，配置会立即应用到预览画布上！

## ⚡ 高级功能

### 对话式调整
你可以在生成后继续对话来微调：

1. 首次生成：
   ```
   我想要一只酷酷的小狗
   ```

2. 然后调整：
   ```
   把背景改成星空
   ```

3. 继续微调：
   ```
   给它加个牛仔帽
   ```

### 组合建议
AI 会智能理解你的描述并选择配件。你可以：

- **指定单个配件**："给它戴个皇冠"
- **指定风格**："未来科技风"
- **指定情绪**："看起来很开心"
- **混合描述**："戴牛仔帽的优雅小狗"

## 🔧 常见问题

**Q: 如何设置 API Key？**  
A: 在 `ai-generator.js` 文件的第 6 行设置，详见 `API_KEY_SETUP.md`。

**Q: API Key 安全吗？**  
A: API Key 内置在代码中。如果是个人使用没问题，但不建议在公开网站使用。

**Q: 会产生费用吗？**  
A: 是的，每次对话会调用 OpenAI API，费用很低（约 $0.001-0.003/次）。

**Q: 可以撤销 AI 的选择吗？**  
A: 可以！你随时可以在右侧面板手动选择其他配件。

**Q: 对话历史会保存吗？**  
A: 对话历史在会话期间保存，刷新页面会重置。

**Q: 需要联网吗？**  
A: 是的，需要网络连接来访问 OpenAI API。
