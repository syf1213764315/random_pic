## 💡 使用技巧

### 1. 从简单开始
不需要一次性描述所有细节，可以从基本风格开始，然后逐步调整。

### 2. 使用关键词
AI 能识别的关键词包括：
- **风格**：酷、可爱、优雅、搞笑、专业、未来、复古
- **情绪**：开心、兴奋、冷酷、温柔
- **场景**：派对、工作、度假、探险
- **配件名称**：牛仔帽、太阳镜、皇冠、气球等

### 3. 描述完整性
虽然简单描述也能工作，但更详细的描述会让 AI 有更好的理解：

**简单**：
```
酷狗
```

**更好**：
```
一只戴太阳镜的酷狗
```

**最好**：
```
一只戴着太阳镜和牛仔帽的酷狗，背景是星空
```

### 4. 主题建议
你可以使用主题词来快速生成特定风格：
- **西部牛仔主题**
- **未来科技主题**
- **皇家贵族主题**
- **派对庆典主题**
- **魔法幻想主题**
- **街头潮流主题**

## 🎨 配件组合建议

### 经典组合

**西部牛仔**
- 头饰：牛仔帽
- 眼睛：太阳镜
- 背景：草地

**未来战士**
- 头饰：头戴式耳机
- 眼睛：激光眼
- 背景：星空

**优雅贵族**
- 头饰：皇冠
- 颈部：蝴蝶结或项圈
- 身体：外套或连衣裙

**派对达人**
- 手持：气球
- 嘴部：微笑
- 背景：烟花

**魔法师**
- 头饰：巫师帽
- 眼睛：星星眼
- 背景：星空
