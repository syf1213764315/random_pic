====================================
   🐕 AI 小狗生成器 - 使用说明
====================================

📋 快速设置（仅需一次）
------------------------------------

第 1 步：获取 OpenAI API Key
  → 访问：https://platform.openai.com/api-keys
  → 登录并创建新的 API Key
  → 复制生成的 Key（格式：sk-proj-xxx...）

第 2 步：设置到项目中
  → 打开文件：ai-generator.js
  → 找到第 6 行：apiKey: 'YOUR_OPENAI_API_KEY_HERE',
  → 替换为你的 API Key：apiKey: 'sk-proj-xxx...',
  → 保存文件

第 3 步：开始使用
  → 在浏览器中打开 index.html
  → 点击 "🤖 AI生成" 按钮
  → 输入描述，例如："我想要一只戴牛仔帽的酷狗"
  → 查看 AI 自动生成的小狗！

====================================

💡 使用示例
------------------------------------

简单描述：
  "戴牛仔帽的小狗"

详细描述：
  "我想要一只看起来很酷的小狗，戴着太阳镜和牛仔帽，
   嘴里露出鲨鱼牙齿，背景是星空"

风格导向：
  "创建一只未来科技风格的小狗"
  "给我一只优雅的小狗"
  "我想要一只派对达人风格的小狗"

====================================

⚠️ 重要提示
------------------------------------

1. API Key 安全
   - API Key 设置在代码中，适合个人使用
   - 不建议将代码发布到公开网站
   - 建议设置 OpenAI 账户的使用限额

2. 费用控制
   - 每次对话约 $0.001-0.003
   - 建议设置月度预算上限
   - 在 OpenAI 后台可以查看使用情况

3. 网络要求
   - 需要连接到互联网
   - 需要能访问 OpenAI API（某些地区可能需要代理）
